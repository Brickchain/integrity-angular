To use the Integrity theme and components you must first install the Material Design components for Angular. Just follow these installation instructions. 
npm install --save @brickchain/integrity-theme
npm install --save @brickchain/integrity-angular
 Import the NgModule for each component you want to use: 
import {{ '{' }}...{{ '}' }} from '@brickchain/integrity-angular';

@NgModule({{ '{' }}
  ...
  imports: [...],
  ...
{{ '}' }})
export class ControllerAppModule {{ '{' }} {{ '}' }}
 Import ~@brickchain/integrity-theme/material-theme in your global stylesheet.
