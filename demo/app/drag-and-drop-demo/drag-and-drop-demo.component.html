The integrityDragAndDrop is an attribute directive that makes any DOM element a receiver of dropped files.

A list of extensions accepted can be specified using the extensions attribute.

If the includeDataURL attribute is set to true, the file objects will also have a dataURL that contains their contents.

There are also emitter attributes for files dropped and files ignored.

Tip: if you include an input of type file inside the DOM element, clicking on the element will open a file dialog.

When hovering over the DOM element the directive will add the class dnd-hover to it.

 
<div
  integrityDragAndDrop
  [extensions]="['png', 'jpg', 'gif', 'svg']"
  [includeDataURL]="false"
  (filesDropped)="showDroppedFiles($event)"
  (filesIgnored)="showIgnoredFiles($event)">
    Drop files here!
    <input type="file" />
  </div>
Attribute Binding Description
integrityClipboard Selector Indicates that DOM element can have files dropped on it.
extensions Input Array of file extensions to accept
includeDataURL Input Indicate if the files should have their contents loaded into a dataURL property
filesDropped Output Callback that receives dropped files with the right extension.
filesIgnored Output Callback that receives dropped files that are ignored (wrong extension).
