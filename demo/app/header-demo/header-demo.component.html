Attribute Required Value Description
title no text The title
withBackButton no - Usage: 'withBackButton' as a property on the element. Consume the backButtonClicked event in a parent component, e.g. (backButtonClicked)="doSomething($event)"
withRightButton no - Usage: 'withRightButton' as a property on the element. Consume the rightButtonClicked event in a parent component, e.g. (rightButtonClicked)="doSomething($event)"
rightButtonIcon no text Usage: rightButtonName="icon-name", where icon name is an angular material icon. If no icon is specified then the settings icon is used.
