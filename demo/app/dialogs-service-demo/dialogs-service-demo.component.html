The DialogsService is a utility that lets you create simple dialogs and handle their results as promises. The service can create Confirmation dialogs, Simple innput dialogs, and QR Code dialogs.

To use the service start by importing the importing the dialogs module in the app module.

Displays a simple dialog with title, message and a confirm/cancel buttons for the user. The result is handled in a promise indicated if the user confirmed or not.

Option Required Description
title no Dialog title
message yes Confirmation message
ok no Label on confirm button
cancel no Label on cancel button
okColor no primary, accent, or warn
cancelColor no primary, accent, or warn
okIcon no Icon on confirm button
cancelIcon no Icon on cancel button

Displays a simple input dialog with title, message, input field and a ok/cancel buttons for the user. The input value is handled as string a promise.

Option Required Description
title no Dialog title
message yes Confirmation message
value no Value of the input field
ok no Label on confirm button
cancel no Label on cancel button
okColor no primary, accent, or warn
cancelColor no primary, accent, or warn
okIcon no Icon on confirm button
