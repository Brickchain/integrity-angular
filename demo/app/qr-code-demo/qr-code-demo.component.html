The <integrity-qrcode> defines a QR code in an HTML page. The QR code is rendered as an SVG image. 
<integrity-qrcode
  qrdata="Hello World"
  level="M">
</integrity-qrcode>
Attribute Required Value Description
qrdata yes text Text contents of the rendered QR code
level no text Error correction level. One of L, M, Q, H. Default M.
M L Q H
