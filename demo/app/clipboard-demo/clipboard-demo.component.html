The integrityClipboard is an attribute directive that makes any DOM element set a value in the OS clipboard upon click.

The value of the directive is what becomes stored in the clipboard.

There are also emitter attributes for success and error events upon click

 
<div
  integrityClipboard="Hello World!"
  (copySuccess)="showCopySuccess($event)"
  (copyError)="showCopyError($event)">Click to copy!</div>
Attribute Binding Description
integrityClipboard Input Value to be stored in the clipboard
copySuccess Output Callback upon success. Receives the copied value as an arg.
copyError Output Callback upon error. Receives the error as an arg.
