The <integrity-schedule> component is used for picking schedules. 
<integrity-schedule
  (scheduleChange)="onScheduleChanged($event)">
</integrity-schedule>
 
    interface ISchedule {{ '{' }}
      from: Date;
      to?: Date;
      until?: Date;
      repeatType?: string;
      period?: number;
      weekdays?: Array<string>;
    {{ '}' }}
Attribute Required Value Description
schedule Input An optional initial schedule can be provided.
scheduleChange Output Any time an input is changed, as well as on initialization, the currently chosen schedule is emitted.
