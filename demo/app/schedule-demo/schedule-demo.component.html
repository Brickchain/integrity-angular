The <integrity-schedule> component is used for picking schedules. 
<integrity-schedule
  (scheduleChange)="onScheduleChanged($event)">
</integrity-schedule>
 
    interface ISchedule {{ '{' }}
      from: Date;
      to?: Date;
      until?: Date;
      repeatType?: string;
      period?: number;
      weekdays?: Array<string>;
    {{ '}' }}
Attribute Required Value Description
schedule no ISchedule Optional initial schedule.
scheduleChange yes ISchedule Any time an input is changed, as well as on initialization, the currently chosen schedule is emitted.
simple no none Toggles a simplified date picker with inputs for 'from' and 'to'.
